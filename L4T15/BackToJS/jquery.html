jQuery is a fast and lightweight JavaScript library that simplifies tasks like HTML document traversal and manipulation, event handling, and animation. It provides a set of functions and methods to make it easier to work with HTML documents.
Functions: Functions in jQuery are methods or actions that can be performed on HTML elements. Examples include after(), append(), before(), empty(), prepend(), text(), val(), and remove().
Selectors: Selectors in jQuery are patterns used to select and manipulate HTML elements. They allow you to target specific elements on a webpage.
Creating and Appending Elements: jQuery provides functions like after(), append(), before(), and prepend() to manipulate the DOM by creating and adding new elements.
Events in jQuery: jQuery simplifies event handling. Common events include mouse events (click, double click, mouse enter), keyboard events (key press, key down, key up), and form events (submit, change, blur).
Effect Chaining: Effect chaining in jQuery allows you to apply multiple effects to an element in a single statement. For example, you can chain effects like hide(), show(), fade(), slide(), animate(), and stop().

built-in functions in jQuery

  • after() - Inserts content after the selected HTML element
  • append() - Inserts content at the end of the selected element
  • before() - Inserts content after the selected HTML element
  • empty() - Removes the child elements from the selected element, but not the element itself
  • prepend() - Inserts content at the beginning of the selected element
  • text() - sets or returns the text content of the selected element
  • val() - Sets or returns the value of a form field
  • remove() - Removes the selected element, including its child elements

Effect chaining:

l$("button").css("color", "blue").slideDown(500).slideUp(500); // example of effect chaining

Effects in jQuery:

  • Hide - Hides the selected element
  • Show - Reveals the selected element
  • Fade - Fades the element in or out
  • Slide - Slides the element
  • Animate - Animates some property of the element
  • Stop - Stops any effect of an element
